## About

A URL shortener that provides a clean web interface for managing shortened URLs. Offers control on the redirect URL name and expiration date. Tracks link usage, dashboarding the resulting data.

## Technology

- Python
  - Flask
  - SQLAlchemy
  - LibSCSS
- PostgreSQL
- `docker-compose`
